WeatherSphereApp

Этот проект позволяет искать информацию о погоде в любом городе мира, используя API OpenWeather. В дополнение к погодным данным, приложение также предоставляет информацию о городе через API Wikipedia, что делает его удобным инструментом для получения полного представления о выбранном месте.

Актуальные погодные данные

OpenWeather — это популярный и надежный сервис, предоставляющий данные о погоде в режиме реального времени для городов по всему миру. Используя их API, это приложение позволяет получать точную и актуальную информацию о погодных условиях в любом городе. Вы можете узнать температуру, скорость ветра и другие параметры.

Введите название интересующего вас города (на английском языке) и получите подробный прогноз на ближайшее время.

Информация о городе

В дополнение к погодным данным, приложение предоставляет краткое описание города через API Wikipedia. Узнайте больше об истории и интересных фактах о выбранном вами месте.

Кроме того, проект включает интеграцию с картой от OpenWeather. Эта карта отображает не только географическое положение выбранного города, но и визуализирует погодные условия в реальном времени.

Удобство использования и современные технологии

Проект разработан с использованием Django, одного из самых популярных веб-фреймворков на Python. Для работы с данными используется Django REST Framework (DRF), который обеспечивает быстрый и эффективный доступ к данным через API.

Также проект включает JavaScript-скрипт, который обеспечивает динамическое обновление интерфейса и взаимодействие с API OpenWeather и Wikipedia

В проекте также применяются современные веб-технологии, такие как GSAP (GreenSock Animation Platform), что позволяет добавить плавные анимации и улучшить взаимодействие с пользователем.
